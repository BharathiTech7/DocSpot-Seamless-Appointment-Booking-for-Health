
# 🩺 DocSpot – Full-Stack Doctor Appointment Booking System

DocSpot is a professional MERN stack application that streamlines the process of booking medical appointments online. Patients can register, browse doctors, book appointments, and upload medical documents. Doctors manage their schedules and appointment statuses, while admins oversee the platform.

## 🚀 Tech Stack

- **Frontend:** React.js, Axios, Bootstrap, Material UI, Ant Design
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT, bcryptjs
- **File Uploads:** Multer
- **Date Handling:** Moment.js

---

## ✨ Features

✅ User registration and login  
✅ Role-based access (Patient, Doctor, Admin)  
✅ Doctor browsing and filtering  
✅ Secure appointment booking  
✅ Document upload for appointments  
✅ Appointment status updates (Confirm, Complete, Cancel)  
✅ Notifications and reminders  
✅ Admin approval for doctor registrations
